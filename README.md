# Employee Management System

## 📌 Description
This is a Java-based console application to manage employee records.

## 🚀 Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## 🛠️ Technologies Used
- Java
- Collections (ArrayList)

## ▶️ How to Run
1. Open terminal
2. Go to src folder
3. Compile:
   javac *.java
4. Run:
   java Main
